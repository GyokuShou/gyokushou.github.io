从零开始复习C语言（1）-数据类型-基本类型

发布 : 2019-05-26 分类 : 笔记

1 数据的表现形式

在计算机高级语言中，数据有两种表现形式：

  • 常量
  • 变量

常量为不可变的量，通常为自然数或计算机符号，可以直接使用。

变量为可变量，须先声明，后使用。

2 常量

常量分为以下5种：

2.1 整型常量

整数

2.2 实型常量

  1. 十进制小数

  2. 指数

    在C语言中，指数使用 e 或 E 代表指数。e/E 前必须有数字，e/E 后必须为整数。

2.3 字符常量

由 '' 单引号括起来的一个字符。

  1. 普通字符

    A, b, # 都是字符常量，也可以用 ASKII 码的数字表示。

  2. 转义字符

    使用 \ 开头的字符为转义字符，常用转义字符见下表。

    | 转义字符 | 字符值 | 输出结果 |
    | ——– | —— | ——– |
    | \' | 单引号 | ' |
    | \" | 双引号 | " |
    | \? | 问号 | ? |
    | \\ | 反斜线 | \ |
    | \a | 警告 | 产生声音或视觉信号 |
    | \b | 退格 | 删除前一个字符 |
    | \f | 换页 | 当前位置移至下页开头 |
    | \n | 换行 | 当前位置移至下行开头 |
    | \r | 回车 | 当前位置移至本行开头 |
    | \t | 水平制表符 | 当前位置移至下一个tab位置 |
    | \0 | 与该8进制码对应的ASKII字符 | |
    | \x | 与该16进制码对应的ASKII字符 | |

2.4 字符串常量

由 "" 双引号括起来的若干个字符，双引号内可以是一个字符。

2.5 符号常量

使用 #define 指令，指定一个符号代称一个常量。

#define PI 3.14
符号常量不是变量，不占内存，只是一个临时符号，用于替换，在预编译后结束生命周期。

3 变量

在编译过程中，编译器会为变量分配一个地址，从变量中取值实际上就是通过变量名找到相对应的内存地址，从该内存地址对应的存储单元中取值。

3.1 常变量

C99允许使用常变量。

常变量有类型，占内存，不可变。

3.2 标识符

对变量，符号常量名，函数，数组，类型等命名的有效字符序列统称表标识符。即对象的名称。

C语言规定标识符只能由数组字母下划线组成，且不允许以数字开头。

同样，命名时应避开关键字。

C语言严格区分大小写，所以 a 和 A 是两个不同的变量。

3.3 变量数据类型

带 * 为C99新增类型。

3.4 基本类型

3.4.1 整型

3.4.1.1 基本整型

基本整型 int 通常为2或4个字节，由编译器决定。

以4字节为例，在内存中存储一个整型数据 5 为：
00000000 00000000 00000000 00000101 形式。

如果进行取反操作如 -5 ，即首先获取 5 的补码，而后对值取反，可得：
11111111 11111111 11111111 11111010 。

再对所得结果 +1 即可得到 -5 的补码：
11111111 11111111 11111111 11111011 。

最左一位为 0 表示正数，最左一位为 1 表示负数。

3.4.1.2 短、长、双长整型

  1. 短整型
    short int 或 short ，通常为2个字节。

  2. 长整型
    long int 或 long ，通常为4个字节。

  3. 双长整型
    long long int 或 long long ，C99新增类型，一般分配8个字节。

3.4.1.3 整型的基本说明

  1. 字节
    1字节存储8位，在4字节整型类型时，可以存储的最大值为：
    01111111 11111111 11111111 11111111 即 2**31-1
    最大容纳范围是：-2^31~2^31-1 。

  2. 类型长度
    C标准没有明确规定各种类型占用存储单元的长度，只需要符合以下规则即可。
    sizeof(short) ≤ sizeof(int) ≤ sizeof(long) ≤ sizeof(long long)

  3. 去除符号
    使用修饰符 unsigned 去除符号，使用unsigned后变量不可存储负数，取值范围为原最大值的2倍。

    只有整型才可以使用 signed 或 unsigned 进行修饰。
    无符号整型使用 %u 输出。

3.4.2 浮点型

由于小数点的位置可以浮动，所以实数的指数形式称为浮点数。

以 3.1415926 为例，可以看做 0.31415926*10^1 即 0.31415926e001，称为规范化指数形式。

C标准没有规定指数部分和小数部分的具体范围，但是可以肯定的是：小数部分占位越多，数值越精确；指数部分占位越多，能表示的数值范围越大。

浮点数分为3种，float、double 和 long double 。

C语言中的浮点型常量，默认采用 double 类型。

在浮点常量后添加 f 如 3.14f 即按照 float 类型处理，编译时不弹出警告。
在浮点常量后添加 l 或 L 如 3.14L 即按照 long double 类型处理。

3.4.3 字符型

字符型以整数形式表示 ASKII 码值，以字符表示实际字符。

如果将一个负数赋值给字符型变量是合法的，如 signed char c = -5; 。如果在定义变量时不添加 signed 或 unsigned ，处理结果由编译器决定。

有符号字符类型实际有效范围为 0~127 。
无符号字符类型范围为 0~255 。
